# MindSeeds

App per sessioni di **binaural beats** e **respirazione guidata**, ottimizzata per desktop e mobile.

## Stack

- React (Vite)
- Tailwind CSS v4
- Tone.js (audio binaurale)
- Framer Motion (animazioni)
- Lucide React (icone)

## Avvio

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
